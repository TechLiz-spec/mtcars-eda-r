# mtcars-eda-r
Exploratory Data Analysis of the MTCARS Dataset

Project Overview

This project presents an exploratory data analysis (EDA) of the built-in mtcars dataset using R. The analysis investigates how different vehicle characteristics influence fuel efficiency and engine performance through data visualization and summary statistics.

The project was developed using Quarto, R, tidyverse, and ggplot2 to demonstrate a complete data analysis workflow.

Objectives

The analysis seeks to answer the following questions:

Does vehicle weight affect fuel efficiency?
Does horsepower affect fuel efficiency?
Do manual cars have better fuel efficiency than automatic cars?
Which cylinder category has the highest average horsepower?
How are cars distributed according to the number of cylinders?
What relationships exist among the numerical variables?
Dataset

The analysis uses the built-in mtcars dataset available in R.

The dataset contains information on 32 automobiles and includes variables such as:

Miles per gallon (mpg)
Number of cylinders (cyl)
Horsepower (hp)
Vehicle weight (wt)
Transmission type (am)
Engine displacement (disp)
Rear axle ratio (drat)
Quarter-mile time (qsec)

Key Findings

Heavier vehicles generally have lower fuel efficiency.
Cars with higher horsepower tend to consume more fuel.
Manual transmission cars generally achieve higher fuel efficiency than automatic cars.
Eight-cylinder vehicles have the highest average horsepower.
Strong positive and negative correlations exist among several numerical variables.
